前言

最优化理论，这是一门相当重要的专业理论必修课。这门课不仅涉及高数，线代，运筹学等知识，更是在实际应用场景中应用广泛（对于算法设计的作用也不小）。同时这门课没有课件，到我们是最后一届开设，同时期末是37开的恐怖比例，逼得我不得不去进行一波博文撰写的提前。

对于写博文这种方式其实算是有利有弊。先说一下弊端吧。无法在课上及时记录，对数学公式的撰写难度较大，同时无法像平板笔记一样图文自由编排。但好处也是显而易见的，可以随时查看，方便复习，同时可以随时修改，方便纠错。同时还有最大的一点优势就是在于md格式对于AI来说是最为友好的文本格式，我完全可以利用各种各样的AI去辅助我进行学习，并将学习的成果进行整理展示，这是独属于这个AI时代的优势。

4. 匈牙利算法思路

指派问题的高效解法——匈牙利算法步骤：

步骤 1：行约简

  • 每行减去该行的最小值

步骤 2：列约简

  • 每列减去该列的最小值

步骤 3：试指派

  • 在零元素上进行试指派，寻找独立零元素
  • 若独立零元素数 = n，得到最优解；否则继续

步骤 4：画盖零线

  • 用最少的线覆盖所有零元素
  • 在未被覆盖的元素中找到最小值进行调整

步骤 5：重复

  • 返回步骤 3，直到找到 n 个独立零元素

4. 投资组合理论要点

有效前沿（Efficient Frontier）：

  • 在给定风险水平下，能够提供最高预期收益的投资组合集合
  • 在给定预期收益水平下，能够提供最低风险的投资组合集合

资本配置线（CAL）：

  • 从无风险利率出发，与有效前沿相切的直线

1. 最优化问题的三要素

约束条件 $g_i(\mathbf{x}) \leq 0, h_j(\mathbf{x}) = 0$ 问题必须满足的限制 产量约束、销量约束、预算约束

2. 最优化问题的建模步骤

解决实际最优化问题的一般流程：

步骤 1：问题分析

  • 明确问题的目标（最大化收益 or 最小化成本）
  • 识别问题的约束条件（资源限制、需求满足等）
  • 确定决策变量（可控制的因素）

步骤 2：建立数学模型

  • 定义决策变量及其取值范围
  • 建立目标函数（用数学表达式描述优化目标）
  • 列出所有约束条件（等式或不等式）

步骤 3：模型分类与选择算法

  • 根据目标函数和约束的形式分类（LP、IP、QP、NLP 等）
  • 选择适当的求解算法

步骤 4：求解与验证

  • 使用解析方法或数值方法求解
  • 验证解的可行性和最优性
  • 灵敏度分析（分析参数变化对解的影响）

3. 最优化问题的分类

根据问题的不同特征，最优化问题可以分为以下几类：

按目标函数和约束的线性性分类：

类型 目标函数 约束条件 例题 求解难度
线性规划 (LP) 线性 线性 运输问题 ⭐⭐
整数规划 (IP) 线性 线性 + 整数 钢管用料 ⭐⭐⭐
二次规划 (QP) 二次 线性 投资优化 ⭐⭐⭐
非线性规划 (NLP) 非线性 非线性 非线性方程组 ⭐⭐⭐⭐

按变量的连续性分类：

  • 连续优化：变量可取实数范围内的任意值（如投资比例）
  • 离散优化/组合优化：变量只能取离散值（如 0-1 变量、整数）

按约束的存在性分类：

  • 无约束优化：$\min f(\mathbf{x})$，没有任何约束
  • 等式约束优化：只有 $h_j(\mathbf{x}) = 0$ 约束
  • 不等式约束优化：包含 $g_i(\mathbf{x}) \leq 0$ 约束

4. 求解方法概述

针对不同类型的问题，有不同的求解策略：

解析方法（适用于简单问题）：

  • 微积分法：令梯度为零，求解方程组
  • 拉格朗日乘数法：处理等式约束
  • KKT 条件：处理不等式约束

数值方法（适用于复杂问题）：

  • 直接搜索法：不依赖导数信息，如单纯形法、模式搜索
  • 梯度法：利用一阶导数信息，如最速下降法、共轭梯度法
  • 牛顿法：利用二阶导数信息，收敛速度快
  • 启发式算法：如遗传算法、模拟退火、粒子群算法（适用于大规模或 NP-hard 问题）

凸函数

凸函数（Convex Function）直观理解就是”碗状”的函数——任意两点间的连线都在函数图像的上方。

数学定义：对于任意 $x_1, x_2$ 和 $\lambda \in [0, 1]$，满足

  • 绿色曲线（$f(x)=x^2$）：凸函数，弦在函数上方，有唯一全局最小值
  • 红色曲线（$f(x)=-x^2$）：凹函数，有唯一全局最大值

凸优化中的重要性质

在凸优化（Convex Optimization）中，有一个非常重要的性质：

凸优化中，局部极值点就是全局极值点。

这是凸优化问题比一般的非凸优化问题更容易求解的重要原因之一。对于凸函数：

  • 若存在极小值点，则该点必为全局最小值点
  • 驻点（梯度为零的点）若存在，则必为全局最优解

标准形的三个要求

要求 说明
1. 目标函数最大化 必须是 $\max Z$，而非 $\min Z$
2. 约束条件等式化 必须是 $=$，而非 $\leq$ 或 $\geq$
3. 变量与资源非负 $x_j \geq 0$，$b_i \geq 0$

$b_i < 0$ 两侧同乘 $-1$ 不等号反向
$x_k \leq 0$ 令 $x_k’ = -x_k$ 替换为非负变量
$x_k$ 无约束 令 $x_k = x_k’ - x_k’’$ 分解为两个非负变量之差

约束转换的直观理解：

  • 松弛变量：表示”还有剩余”的量（$\leq$ 变 $=$，加上松弛）
  • 剩余变量：表示”还差多少”的量（$\geq$ 变 $=$，减去剩余）
