《数据分析方法（第二版）》重要概念

发表于2025-05-18|更新于2025-05-18
|总字数:3.1k|阅读时长:11分钟|浏览量:

一、数据描述性分析

（一）数据特征

1. 集中趋势

知识点 定义式 相近知识点 对比说明
均值 对于一组数据 (x1,x_2,cdots,x_n)，均值 (ar{x}= rac{1}{n}sum{i = 1}^{n}x_i) 算术平均数、加权平均数 算术平均数是均值的常见形式；加权平均数考虑了各数据的权重，公式为 (ar{x}= rac{sum{i = 1}^{n}w_ix_i}{sum{i = 1}^{n}w_i})，其中 (w_i) 为权重。均值易受极端值影响，而加权平均数可根据实际情况调整各数据的重要性
中位数 将数据按从小到大排序后，若 (n) 为奇数，中位数 (M = x{( rac{n + 1}{2})})；若 (n) 为偶数，中位数 (M= rac{x{( rac{n}{2})}+x_{( rac{n}{2}+1)}}{2}) 分位数 分位数是中位数的推广，如 (p) 分位数 (Mp)，当 (np) 不是整数时，(M_p=x{([np]+1)})；当 (np) 为整数时，(Mp= rac{1}{2}(x{(np)}+x_{(np + 1)}))。中位数不受极端值影响，能较好地反映数据的中心位置
众数 数据中出现次数最多的数值 无 众数可用于描述定性数据的集中趋势，可能不存在或有多个，不受极端值影响

2. 离散程度

知识点 定义式 相近知识点 对比说明
方差 样本方差 (s^2= rac{1}{n - 1}sum_{i = 1}^{n}(x_i-ar{x})^2) 标准差 标准差 (s=sqrt{ rac{1}{n - 1}sum_{i = 1}^{n}(x_i-ar{x})^2})，是方差的算术平方根。方差和标准差都衡量数据的离散程度，标准差的单位与原始数据相同，更具直观性
极差 (R = max(x_i)-min(x_i)) 四分位距 四分位距 (IQR = Q_3 - Q_1)，其中 (Q_1) 为下四分位数，(Q_3) 为上四分位数。极差计算简单，但对极端值敏感；四分位距能排除部分极端值的影响，更稳健地反映数据的离散程度
变异系数 (CV= rac{s}{ar{x}} imes100%) 无 变异系数用于比较不同数据集的离散程度，消除了数据量纲和均值的影响

3. 分布形状

知识点 定义式 相近知识点 对比说明
偏度 (SK= rac{n}{(n - 1)(n - 2)}sum_{i = 1}^{n}( rac{x_i-ar{x}}{s})^3) 无 偏度衡量数据分布的不对称程度，(SK = 0) 时分布对称；(SK>0) 时，分布呈正偏态；(SK<0) 时，分布呈负偏态
峰度 (K= rac{n(n + 1)}{(n - 1)(n - 2)(n - 3)}sum_{i = 1}^{n}( rac{x_i-ar{x}}{s})^4- rac{3(n - 1)^2}{(n - 2)(n - 3)}) 无 峰度描述数据分布的尖峰或扁平程度，与正态分布相比，(K>0) 时，分布更尖峭；(K<0) 时，分布更扁平

（二）相关性分析

1. 相关系数

知识点 定义式 相近知识点 对比说明
皮尔逊相关系数 (r= rac{sum{i = 1}^{n}(x_i-ar{x})(y_i-ar{y})}{sqrt{sum{i = 1}^{n}(xi-ar{x})^2sum{i = 1}^{n}(y_i-ar{y})^2}}) 斯皮尔曼等级相关系数 斯皮尔曼等级相关系数是基于数据的秩次计算的，适用于非正态分布或顺序数据；皮尔逊相关系数适用于两个连续变量的线性相关分析

（三）数据分布

1. 常见分布类型

知识点 定义式 相近知识点 对比说明
正态分布 概率密度函数 (f(x)= rac{1}{sqrt{2pi}sigma}e^{- rac{(x-mu)^2}{2sigma^2}})，其中 (mu) 为均值，(sigma) 为标准差 均匀分布、二项分布、泊松分布等 均匀分布中每个值出现的概率相同；二项分布描述 (n) 次独立伯努利试验中成功的次数；泊松分布描述在固定时间或空间内事件发生的次数。正态分布是最常见的连续分布，许多自然和社会现象近似服从正态分布

（四）均值比较

1. 均值分析与比较均值

知识点 定义式 相近知识点 对比说明
均值分析 计算一组数据的均值 (ar{x}= rac{1}{n}sum_{i = 1}^{n}x_i) 无 均值分析用于描述数据的集中趋势，可判断得分最高和最低的方面，有助于把握整体情况
比较均值 按指定条件对样本分组计算均值并比较 无 比较均值可判断不同数据组是否有差异，有助于对比各类分组的差异

二、回归分析

（一）线性回归模型

1. 简单线性回归

知识点 定义式 相近知识点 对比说明
回归方程 (y=eta_0+eta_1x+epsilon)，其中 (eta_0) 为截距，(eta_1) 为斜率，(epsilon) 为误差项 无 简单线性回归用于研究一个自变量 (x) 与一个因变量 (y) 之间的线性关系
参数估计 最小二乘法估计 (hat{eta}1= rac{sum{i = 1}^{n}(xi-ar{x})(y_i-ar{y})}{sum{i = 1}^{n}(x_i-ar{x})^2})，(hat{eta}_0=ar{y}-hat{eta}_1ar{x}) 无 通过最小二乘法使误差平方和最小，得到回归系数的估计值

2. 多元线性回归

知识点 定义式 相近知识点 对比说明
回归方程 (y=eta_0+eta_1x_1+eta_2x_2+cdots+eta_kx_k+epsilon) 无 多元线性回归研究多个自变量 (x_1,x_2,cdots,x_k) 与一个因变量 (y) 之间的线性关系
参数估计 矩阵形式 (hat{eta}=(X^TX)^{-1}X^Ty)，其中 (X) 为自变量矩阵，(y) 为因变量向量 无 通过矩阵运算求解回归系数的估计值

（二）Logistic回归模型

1. Logistic回归方程

知识点 定义式 相近知识点 对比说明
回归方程 (P(Y = 1 X)= rac{1}{1 + e^{-(eta_0+eta_1x_1+eta_2x_2+cdots+eta_kx_k)}}) 无 Logistic回归用于处理二分类问题，将线性组合通过Logistic函数转换为概率值
参数估计 最大似然估计 无 通过最大化似然函数来估计模型参数

三、方差分析

（一）单因素方差分析

1. 模型与检验

知识点 定义式 相近知识点 对比说明
模型 (x{ij}=mu+alpha_i+epsilon{ij})，其中 (x{ij}) 为第 (i) 组第 (j) 个观测值，(mu) 为总均值，(alpha_i) 为第 (i) 组的效应，(epsilon{ij}) 为误差项 无 单因素方差分析研究一个因素的不同水平对因变量的影响
F检验 (F= rac{MSA}{MSE})，其中 (MSA) 为组间均方，(MSE) 为组内均方 无 通过F检验判断因素的不同水平对因变量是否有显著影响

（二）两因素方差分析

1. 等重复试验下的方差分析

知识点 定义式 相近知识点 对比说明
模型 (x{ijk}=mu+alpha_i+eta_j+(alphaeta){ij}+epsilon{ijk})，其中 (x{ijk}) 为第 (i) 个因素水平、第 (j) 个因素水平下的第 (k) 个观测值，(mu) 为总均值，(alphai) 为因素 (A) 的第 (i) 个水平的效应，(eta_j) 为因素 (B) 的第 (j) 个水平的效应，((alphaeta){ij}) 为因素 (A) 和 (B) 的交互效应，(epsilon_{ijk}) 为误差项 无 两因素等重复试验下的方差分析考虑两个因素的主效应和交互效应
F检验 分别对因素 (A)、因素 (B) 和交互效应进行F检验 无 通过F检验判断因素 (A)、因素 (B) 及其交互效应对因变量是否有显著影响

四、主成分分析与典型相关分析

（一）主成分分析

1. 主成分的定义与计算

知识点 定义式 相近知识点 对比说明
总体主成分 设总体 (X=(X_1,X_2,cdots,X_p)^T)，协方差矩阵为 (Sigma)，其特征值 (lambda_1geqlambda_2geqcdotsgeqlambda_pgeq0)，对应的单位特征向量为 (u_1,u_2,cdots,u_p)，则第 (i) 个主成分 (Y_i = u_i^TX) 无 主成分分析通过线性变换将原始变量转换为一组互不相关的主成分，提取数据的主要信息
样本主成分 根据样本数据计算协方差矩阵 (S)，求解 (S) 的特征值和特征向量，得到样本主成分 无 样本主成分是基于样本数据计算的，用于对样本进行分析和解释

（二）典型相关分析

1. 典型变量与典型相关系数

知识点 定义式 相近知识点 对比说明
典型变量 设 (X=(X1,X_2,cdots,X_p)^T) 和 (Y=(Y_1,Y_2,cdots,Y_q)^T) 为两组变量，(Sigma{11})、(Sigma{22}) 分别为 (X) 和 (Y) 的协方差矩阵，(Sigma{12}) 为 (X) 和 (Y) 的协方差矩阵，求解广义特征值问题 ((Sigma{12}Sigma{22}^{-1}Sigma{21}-lambdaSigma{11})a = 0) 和 ((Sigma{21}Sigma{11}^{-1}Sigma{12}-lambdaSigma{22})b = 0)，得到典型变量 (U_i = a_i^TX) 和 (V_i = b_i^TY) 无 典型相关分析研究两组变量之间的相关性，通过寻找两组变量的线性组合，使得它们之间的相关性最大
典型相关系数 典型相关系数为典型变量之间的相关系数 (

ho_i= ext{Corr}(U_i,V_i)) |无|典型相关系数衡量两组典型变量之间的相关性|

五、判别分析

（一）距离判别

1. 两个总体的距离判别

知识点 定义式 相近知识点 对比说明
距离度量 常用欧氏距离 (d(x,mu_i)=sqrt{(x - mu_i)^T(x - mu_i)}) 和马氏距离 (d^2(x,mu_i)=(x - mu_i)^TSigma^{-1}(x - mu_i))，其中 (x) 为样本，(mu_i) 为总体 (i) 的均值，(Sigma) 为总体的协方差矩阵 无 距离判别根据样本到两个总体的距离来判断样本属于哪个总体，马氏距离考虑了变量之间的相关性

（二）Bayes判别

1. Bayes判别准则

知识点 定义式 相近知识点 对比说明
判别准则 (P(G_i x)= rac{P(Gi)f_i(x)}{sum{j = 1}^{k}P(G_j)f_j(x)})，其中 (P(G_i)) 为总体 (G_i) 的先验概率，(f_i(x)) 为总体 (G_i) 的概率密度函数 无 Bayes判别根据样本的先验概率和条件概率，计算样本属于各个总体的后验概率，选择后验概率最大的总体作为样本的归属

六、聚类分析

（一）样品间相近性的度量

1. 距离度量

知识点 定义式 相近知识点 对比说明
欧氏距离 (d{ij}=sqrt{sum{k = 1}^{p}(x{ik}-x{jk})^2})，其中 (x{ik}) 和 (x{jk}) 分别为样品 (i) 和 (j) 的第 (k) 个变量值 曼哈顿距离、切比雪夫距离等 曼哈顿距离 (d{ij}=sum{k = 1}^{p} x{ik}-x{jk} )；切比雪夫距离 (d{ij}=max{1leq kleq p} x{ik}-x{jk} )。不同的距离度量方法适用于不同的数据类型和问题场景

（二）聚类方法

1. 快速聚类法

知识点 定义式 相近知识点 对比说明
步骤 先选择初始聚类中心，然后将样品分配到最近的聚类中心，再更新聚类中心，重复这个过程直到聚类中心不再变化 谱系聚类法 谱系聚类法通过逐步合并相似的类形成聚类结果；快速聚类法更适合处理大规模数据，效率较高

（三）谱系聚类法

1. 类间距离

知识点 定义式 相近知识点 对比说明
最短距离 (D{ij}=min{xin G_i,yin G_j}d(x,y))，其中 (G_i) 和 (G_j) 为两个类，(d(x,y)) 为样品 (x) 和 (y) 之间的距离 最长距离、平均距离等 最长距离 (D{ij}=max{xin Gi,yin G_j}d(x,y))；平均距离 (D{ij}= rac{1}{nin_j}sum{xin Gi}sum{yin G_j}d(x,y))。不同的类间距离定义会导致不同的聚类结果

七、Bayes统计分析

（一）Bayes统计模型

1. Bayes定理

知识点 定义式 相近知识点 对比说明
Bayes定理 (P( heta x)= rac{P(x heta)P( heta)}{P(x)})，其中 (P( heta)) 为先验分布，(P(x heta)) 为似然函数，(P(x)) 为边缘分布，(P( heta x)) 为后验分布 无 Bayes定理将先验信息和样本信息结合起来，得到后验分布，用于对参数进行推断

（二）Bayes统计推断

1. 参数的Bayes点估计

知识点 定义式 相近知识点 对比说明
后验均值估计 (hat{ heta}=E( heta x)=int heta P( heta x)d heta) 后验中位数估计、后验众数估计 后验中位数估计是后验分布的中位数；后验众数估计是后验分布的众数。不同的点估计方法适用于不同的情况
